Privacy Policy

Scope
This policy applies to the Smart Security Gateway service and all protected domains.

Data We Collect
- Behavioral signals: mouse movement, click patterns, typing timing
- Technical data: HTTP headers, request rate, navigation paths
- Session identifiers to link events

Data We Do Not Collect
- Raw form contents or sensitive input fields
- Images or explicit personal data without a security need

Purpose of Collection
- Detect bots and automated attacks
- Improve ML models and reduce false positives
- Provide security reporting to domain owners

Retention
- Behavioral data: default 7 days
- Security reports: default 30 days

Sharing
Data is not sold or shared with third parties unless required by law.

User Rights
Users may request data deletion through the domain owner.

Updates
If the policy changes, the updated version will be published here.
